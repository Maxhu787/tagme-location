## Setup

### app

```
npm install
npm run prebuild
npm run android
```

### database

(需要 Docker)

```
supabase start
```

## 主要功能

- 使用者註冊／登入 (Google 登入)
- 好友即時定位、隱形模式
- 個人檔案 (頭貼、自介、國家等)
- 加好友、好友申請通知

## 使用語言與框架

### 前端

- react native
- expo

### 後端

- supabase
